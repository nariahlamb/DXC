# idea-2: CenterPanel 三层交互闭环

## Concept
建立“主行动（Top5）-次行动（展开）-自由输入”三层交互，减少纯文本输入成本，提高操作确定性。

## Requirements
- 维护高频动作排序策略。
- 选项点击可预填命令草稿。
- 保留自由输入，避免限制策略玩法。

## Risk Matrix
- 操作过度模板化：中 -> 保留自由输入优先权。
- 移动端拥挤：中 -> 分组折叠 + 横滑。

## MVP
1. Top5 快捷行动。
2. 预填 + 一键发送。
3. 最近使用动作回流机制。

## Success Metrics
- 从“看到选项”到“提交动作”耗时下降 25%。
