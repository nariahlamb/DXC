# merged-idea-1: 自适应引导体验层

## Unified Concept
融合“因果时间轴 + 三层交互闭环 + 跨端优先级协议 + 自适应可读性”，形成统一引导体验层。

## Preserved Elements
- 时间轴提供上下文。
- 三层交互缩短执行路径。
- 优先级协议保证跨端一致。
- 可访问性与动效预算保证不同设备可用。

## Tradeoffs
- 暂缓 AI 全自动重排，先做可解释推荐。
- 暂缓全量状态机改造，先用 facade 降耦。

## Implementation Considerations
- 先做数据关联模型，再做 UI 可视层。
- 全程建立性能与可读性指标门禁。
