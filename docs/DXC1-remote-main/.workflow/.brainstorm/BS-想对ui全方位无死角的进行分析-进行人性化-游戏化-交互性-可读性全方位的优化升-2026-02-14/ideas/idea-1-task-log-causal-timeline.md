# idea-1: 任务-日志联动因果时间轴

## Concept
把日志升级为“可决策时间轴”：每条关键日志可定位关联任务、关联行动、回合上下文，并支持关键事件回放入口。

## Requirements
- 新增事件关联模型：eventId / questId / turnId。
- CenterPanel 支持“按任务过滤 + 按回合折叠 + 跳转最新关键日志”。
- TasksModal 点击任务可触发日志过滤。

## Risk Matrix
- 误匹配风险：中 -> 混合规则（ID + 关键词 + 回合区间）。
- 性能风险：中 -> 虚拟滚动/分页渲染。
- 理解成本：低 -> 默认保留“普通日志流”视图。

## MVP
1. 任务过滤 + 回合折叠。
2. 关键日志标记与快速跳转。
3. 任务卡联动入口。

## Success Metrics
- 日志阅读到行动的平均点击次数下降 30%。
- 玩家“找不到下一步”的反馈下降。
