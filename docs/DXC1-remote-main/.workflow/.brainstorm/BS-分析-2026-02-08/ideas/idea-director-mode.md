# Idea Deep-Dive: Director Mode Experience Engine

## 核心概念
将当前“静态动作按钮”升级为“导演式互动流程”：行动建议、风险可视、结果反馈。

## 目标
- 提升游戏性与沉浸感
- 降低玩家决策门槛
- 保持高自由度

## 设计要点
1. 意图层（攻击/控制/社交/撤退）
2. 执行层（具体行动）
3. 反馈层（掷骰、资源变化、状态变化）
4. 总结层（回合结算摘要）

## 关键指标
- 平均决策时间
- 每回合交互次数
- 玩家选择多样性

## 风险与缓解
- 信息过载 → progressive disclosure
- 过度引导 → 建议强度开关

## MVP
- 风险标签
- 结果预览
- 回合摘要面板

## 决策
**Recommendation**: pursue
